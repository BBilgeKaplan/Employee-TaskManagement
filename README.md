# DevExpress XAF Framework - Employee Task Management  

DevExpress XAF Framework'ü kullanarak çalışan bilgilerini ve görev planlamasını yöneten basit bir proje.  
Yan panelde bulunan bölümler :
- Users
- Departments
- Employees
- Calendar
- Payroll
- Resumes
- Tasks
  
#### Employees  
Çalışanların isim, doğum tarihi, unvan, mail, adres, departman, ofis ve iş pozisyonu gibi bilgilerinin tutulduğu bölümdür.  
Görevler bölümüyle entegre edilerek bir çalışanın bilgilerine girdiğinizde ona atanmış olan tüm görevleri görüntülemenize yardımcı olur.

#### Departments  
Departman isminin, lokasyonunun, ofisinin ve departman sorumlusunun bilgisinin tutulduğu bölümdür.  
Çalışanlar bölümüyle entegre edilerek bu departmana atanmış çalışan bilgileri görüntülenebilmektedir.  

<img src="https://github.com/user-attachments/assets/23dd2df2-988f-4dd6-abae-489cce9df30d" alt="Departman Görseli" width="850" height="">

#### Tasks  
Görevlerin konusu, görevin kime atandığı, başlama ve bitiş tarihleri, görevin önem derecesi, görevin tamamlanma durumu tanımlandı.  
Detay görünümde çalışan sınıfıyla bağlantı kurularak görevlerin hangi çalışanlara atandığı gösterildi.
